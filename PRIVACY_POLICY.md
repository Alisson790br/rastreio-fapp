# Política de Privacidade — Rastreio FAPP

**Última atualização:** 12 de junho de 2026

## 1. Introdução

A extensão **Rastreio FAPP** foi desenvolvida para facilitar a busca e o rastreio de pedidos por número, a sincronização com o SharePoint, a cópia de códigos e links, e o acesso direto à transportadora.

Esta Política de Privacidade descreve como tratamos as informações dos usuários.

## 2. Dados Coletados

A extensão **não coleta, não armazena e não transmite** dados pessoais a servidores externos.

As únicas informações utilizadas são:

- **Dados de rastreio de pedidos** inseridos pelo próprio usuário, armazenados localmente no navegador via `chrome.storage` exclusivamente para funcionamento da extensão.
- - **Acesso ao SharePoint** (`https://*.sharepoint.com/*` e `https://*.sharepointonline.com/*`) apenas para sincronização de pedidos solicitada pelo usuário.
 
  - ## 3. Uso das Permissões
 
  - | Permissão | Finalidade |
  - |---|---|
  - | `storage` | Salvar dados de rastreio localmente no navegador do usuário |
  - | `host_permissions` (SharePoint) | Sincronizar pedidos do SharePoint quando solicitado pelo usuário |
 
  - ## 4. Compartilhamento de Dados
 
  - A extensão **não compartilha** nenhum dado com terceiros. Nenhuma informação é enviada a servidores externos, anunciantes ou qualquer outra entidade.
 
  - ## 5. Segurança
 
  - Todos os dados ficam armazenados localmente no dispositivo do usuário. Não há transmissão de dados a servidores externos.
